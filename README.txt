Hello,

This is an application to help understand the different bootcamps and their information. This is listed in alphabetical order.

To run the application, type the following into the console prior to running the code.

gem install pry
gem install nokogiri

After the gem's have installed, run this into the console to execute the code.

ruby bin/run

Type in 1-35 to expand on the bootcamp's information. Type 0 to exit.
